System Architecture

Monorepo — Hono API + React SPA + MongoDB

Browser

web/

React 19 + Vite + TypeScript

  • Auth pages (login/register)
  • Home — tale library + discovery
  • Workspace — node graph editor
  • Play view — rich player UI
  • Tailwind CSS + Radix UI
Server

api/

Hono + Zod OpenAPI + TypeScript

  • Auth routes (JWT)
  • Tales CRUD + publishing
  • Play sessions + save states
  • AI writing endpoints
  • Swagger UI at /doc
Data & Services

External

MongoDB Atlas + Claude API

  • MongoDB (Mongoose ODM)
  • Claude API (AI writing)
  • JWT secret in env
  • API key stored server-side
Monorepo Layout
storyforge/
├── api/          ← Hono server (Node.js)
│   ├── src/
│   │   ├── routes/   (auth, tales, play, ai)
│   │   ├── models/   (Mongoose schemas)
│   │   └── lib/      (claude client, jwt utils)
│   └── package.json
├── web/          ← React SPA (Vite)
│   ├── src/
│   │   ├── pages/
│   │   ├── components/
│   │   └── lib/      (api client)
│   └── package.json
└── shared/       ← TypeScript types (shared)
    └── types.ts
